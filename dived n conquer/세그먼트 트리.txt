세그먼트 트리? (segment Tree)
배열처럼 데이터가 일렬로 나열되어 있을 때, 
특정 구간의 합, 또는 특정 구간에서 가장 큰 수 등 <구간의 정보>를 효율적으로 얻어내기 위해 설계된 자료구조

1. 기본구조 - 이진트리(Binary Tree)
각 노드 - 구간의 정보
부모 노드 ? 자식 노드의 구간 범위 전체를 포괄
      1 ~ 8
   1~4   5~8

- 최상단 루트  = 전체 구간의 정보
- 최하단 라프 = 크기가 1인 범위 = 데이터 1개

2. 연산
- update
데이터 갱신(추가/수정/제거)
위치 인덱스, 실제 데이터 인자를 이용하여 특정 인덱스 위치에 있는데이터를 갱신하는 작업 수행
- query
구간의 정보를 탐색하여 사용자에게 반환하는 작업 수행
start_idx, end_idx 제공


height = (int)ceil(log2(n))
