동적 계획법
: 처음 진행 된 연산은 기록해 두고, 이미 진행했던 연산이라면 다시 연산하는 것이 아니라 기록되어 있는 값을 가져옴
제를 풀 때 하나의 문제를 여러 하위 문제로 나누어 풀고, 그것들을 결합해서 최종 목적에 도달하는 방식의 알고리즘이에요.

메모이제이션(Memoization)
위의 코드에서는 하위 문제를 해결할 때 그 해결책을 저장해 두고,
똑같은 문제가 발생했을 때 저장되어 있던 해결책을 가지고 간단하게 해결했습니다.
이렇게 동일한 문제를 반복해야 할 경우,
한 번 계산된 결과를 저장해 두었다가 활용하는 방식으로 중복 계산을 줄이는 것을 메모이제이션(Memoization)이라고 합니다.

--> 모든 방법을 일일이 검토하여 최적의 해를 찾아내는 방식의 알고리즘

(동적 계획법 <-> 그리디 알고리즘)

단점) 시간이 오래 걸림
장점) 항상 최적의 해를 구할 수 있음
